# Findings Surface

This directory is the reviewer-facing findings surface for the repository.

Purpose:

- show what currently matters most scientifically
- separate stable or near-stable findings from broader lab documentation
- give external readers a short path into claims, evidence level, and claim boundary

## Current evidence status

- evidence level in the active GPT-2 Small setup: `Supported`
- cross-model status: not yet replicated
- current observer/intervention rule: read-only traces and write-back interventions must be treated
  as distinct evidence classes
