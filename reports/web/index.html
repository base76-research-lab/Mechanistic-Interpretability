Key Finding
High entropy alone does not distinguish hallucination from legitimate reasoning.

The discriminating signal is the combination of gap_state_to_candidates magnitude and candidate_frontier_coherence. Two high-entropy states can diverge sharply: one represents active reasoning, the other a degenerate fallback regime heading toward hallucination.

Risk Signal Definition
Formula
entropy_norm = H / 10.0
gap_norm = gap / (1 + spread_mean)
risk_score = min(1.0, 0.5·entropy_norm + 0.5·gap_norm)
Metric definitions
H— logit entropy on last token (natural log)
gap_state_to_candidates— L2 distance: residual state vs mean(candidate coords)
candidate_spread_mean— mean L2 of top-k W_U token projections
operator_strength— ‖state_coords‖ (triage signal, not in risk formula)
Threats to Validity
SAE-basis dependence

Different seeds may yield different bases. Mitigation: cluster-PCA over single neurons; seeds logged.

Model scale (GPT-2 small)

May not capture phenomena at scale. Mitigation: Phi-2 parallel run (in progress), model version logged.

Small prompt set (3 scenarios)

Preliminary. Mitigation: exp_003 batch (50–100 prompts, robust compression pipeline).
