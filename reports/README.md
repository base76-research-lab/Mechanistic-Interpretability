# Reports / Findings

This folder contains *reviewable* results: what was run, what was observed, and what is actually new.

## Index

- `exp_001_sae.md` — SAE on GPT-2 layer 5 + Field View risk signal (v1-v3)
- `feature_dict.md` — feature dictionary (labels + examples)
- `logs/2026-03-04.md` — run log + changes + next steps
- `figures/` — images used in reports (generated by `scripts/make_figures.py`)

## Report standard (short)

Each findings report should include:
- Setup: model, layer, prompts/dataset, parameters
- Results: metrics (with file paths to JSON/artifacts)
- Observations: *what actually happens* (before interpretation)
- Interpretation: hypothesis, mechanism, predictions
- Threats to validity: what might be an artifact
- Next steps: 1-3 concrete runs

Template: `template_findings.md`
