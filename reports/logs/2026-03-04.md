## Risk and gap definition (Field View)

```text
entropy_norm = H / 10.0
gap_norm     = gap_state_to_candidates / (1 + candidate_spread_mean)
risk_score   = min(1.0, 0.5 * entropy_norm + 0.5 * gap_norm)
```

- `H`: logit entropy for the next-token distribution (natural log).
- `state_coords`: projection of the residual onto the chosen SAE subspace (often PCA/cluster mean).
- `candidate_coords`: projection of top-k W_U token vectors onto the same subspace.
- `gap_state_to_candidates` = ||state_coords - mean(candidate_coords)||_2. (Min-dist is logged, but not used in risk.)
- `candidate_spread_mean`: mean ||candidate_coords||.
- `|coords|` (operator strength): ||state_coords||, used for triage but not yet in the risk formula.

## Threats to validity

- **SAE basis dependence**: different seeds can produce different bases.
  Mitigation: use cluster PCA instead of single units; log seeds.
- **Model size (GPT-2 small)**: may miss phenomena or exaggerate effects.
  Mitigation: run Phi-2 in parallel; log model versions.
- **Prompt-set size (17 prompts)**: limited.
  Mitigation: expand the prompt list; each run saves prompts + checksums.
