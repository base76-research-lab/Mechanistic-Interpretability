# Notes

This folder is for "lab notes": short, timestamped observations that are useful for continuity, but not yet
fully distilled into a findings report.

Rules of thumb:
- Keep one note per topic/day; prefer Markdown.
- Anything that becomes a claim should graduate into [`reports/`](../reports/) with artifacts + repro steps.
- Avoid sensitive data; assume notes are public.

Suggested naming:
- `YYYY-MM-DD_topic.md`
